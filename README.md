# react-native-idfa-aaid

React Native module for getting IDFA (iOS) or AAID (Android)

## Installation

```sh
npm install react-native-idfa-aaid
```

## Usage

```js
import { multiply } from 'react-native-idfa-aaid';

// ...

const result = await multiply(3, 7);
```
